Command Line Options

Start with the embedded H2 Database server​

java -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded -h2

Start using a PostgreSQL JDBC driver​

java -cp postgresql-42.7.3.jar:ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded

Start on port 8000​

Start Web/SOAP interface on port 8000 (instead of 8080)

java -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded -port 8000

Start using IPv4 only sockets​

java -Djava.net.preferIPv4Stack=true -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded

Start with SOAP interface​

java -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded -soap

Start with SOAP interface and without a Web interface​

java -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded -soap -headless

Start with config directory conf/​

By default the config directory is the directory from which you start the ebms-admin. You can change the config directory by setting configDir

java -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded -configDir conf/

Start with a custom log4j2 file log4j2.xml​

java -Dlog4j.configurationFile=log4j2.xml -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded

Start without using the default Java truststore​

java -Djavax.net.ssl.trustStore= -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded

Start with HTTPS​

Start with HTTPS Web/SOAP interface using keystore keystore.p12

java -Djavax.net.ssl.trustStore= -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded \
-ssl -keyStoreType PKCS12 -keyStorePath keystore.p12 -keyStorePassword password

Start with HTTPS and client authentication​

Start with HTTPS Web/SOAP interface using keystore keystore.p12
and require SSL client authentication using truststore truststore.p12 (which holds the client's certificate chain)

java -Djavax.net.ssl.trustStore= -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded \
-ssl -keyStoreType PKCS12 -keyStorePath keystore.p12 -keyStorePassword password \
-clientAuthentication -trustStoreType PKCS12 -trustStorePath truststore.p12 -trustStorePassword password

Start with HTTPS, client authentication and client certifiate authentication​

Start with HTTPS Web/SOAP interface using keystore keystore.p12
and require SSL client authentication using truststore truststore.p12 (which holds the client's certificate chain)
and authenticate client SSL certificate using clientTruststore.p12 (which holds the client's certificate)

java -Djavax.net.ssl.trustStore= -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded \
-ssl -keyStoreType PKCS12 -keyStorePath keystore.p12 -keyStorePassword password \
-clientAuthentication -trustStoreType PKCS12 -trustStorePath truststore.p12 -trustStorePassword password \
-authentication -clientTrustStoreType PKCS12 -clientTrustStorePath clientTruststore.p12 -clientTrustStorePassword password

Start in EbMS Server mode​

java -Djavax.net.ssl.trustStore= -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded -headless -disableEbMSClient

Start in EbMS Client mode​

java -Djavax.net.ssl.trustStore= -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded -headless -disableEbMSServer

Start in SOAP API mode​

java -Djavax.net.ssl.trustStore= -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded -soap -headless -disableEbMSServer -disableEbMSClient

Start in Web Interface mode​

java -Djavax.net.ssl.trustStore= -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded -disableEbMSServer -disableEbMSClient

Start Health service on port 8089​

Start Health service on port 8089 (instead of default port 8008)

java -Djavax.net.ssl.trustStore= -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded -health -healthPort 8089

Start using basic authentication​

Start using basic authentication on Web/SOAP interface

java -cp ebms-admin-2.20.3.jar nl.clockwork.ebms.admin.StartEmbedded -authentication
